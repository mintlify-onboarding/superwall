---
title: "Superwall Events"
---

The SDK automatically tracks some events, which power the charts in the dashboard. 

We encourage you to track them in your own analytics as described in [3rd Party Analytics](/docs/3rd-party-analytics).

The following Superwall events can be used as triggers to present paywalls:

* `app_install`
* `app_launch`
* `deepLink_open`
* `session_start`
* `paywall_decline`
* `transaction_fail`
* `transaction_abandon`
* `survey_response`

For more info about how to use these, check out [Using Implicit Events](/docs/using-implicit-events).
